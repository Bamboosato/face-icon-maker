# Face Icon Maker MVP開発タスク

## 1. 目的

MVPのゴールは以下とする。

```text
集合写真をアップロード
↓
顔を自動検出
↓
対象者を選択
↓
アイコン生成
↓
PNG保存
```

最小機能で価値を提供することを優先する。

## P7-03 動作確認

テスト観点

- [ ] 機能観点: アップロード、顔検出、顔選択、自動クロップ、手動調整、PNG保存
- [ ] 非機能観点: 顔検出開始3秒以内、顔選択100ms以内、PNG生成1秒以内
- [ ] データ観点: 人数、画像向き、画像形式、画像サイズ、顔未検出画像
- [ ] UI観点: スマホ操作、顔枠タップ、選択ハイライト、プレビュー、エラー表示
- [ ] 正常系: 画像選択からPNG保存まで30秒以内で完了
- [ ] 異常系: 不正形式、読込失敗、顔未検出、サイズ超過
- [ ] 境界値: 50MB直前・以上、50MP直前・以上、長辺3000px前後、幅・高さ12000px前後
- [ ] 状態遷移: 戻る、再編集、最初からやり直す、別画像選択
